POSH E-commerce Website

This is a simple Flask-based e-commerce website project that allows users to register, log in, browse products, add them to a shopping cart, and place orders.

Features

User Registration: Users can create a new account by providing a unique username and password.
User Login: Registered users can log in using their credentials.
Product Listing: Users can view a list of available products with their details such as name, price, and description.
Shopping Cart: Users can add products to their shopping cart, adjust quantities, and remove items.
Order Placement: Users can place orders for the products in their shopping cart.
User Management: Admin users can manage user accounts, including editing and deleting user details.
